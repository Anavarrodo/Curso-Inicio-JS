bucle for se utiliza para que se ejecute varias veces algo.
para ello utilizamos la variable for y dentro metemos la variable que queremos inicializar. en este caso es var i = 0 (es el contador).
El segundo paso es ponerle una condicion para que ocurra ese bucle. eso lo hacemos si i es menor que 100 (porque en nuestr ej. numero es 100).
Para que en cada bucle le sume 1 y vaya de incrementando tenemos que utilizar la variable i++

contador	var i = 0 (el contador empieza en 0)
condicion	i < numero (la  inicializacion tiene que ser menor que 100)
incrementador	i++ 	(a cada vuelta del bucle le sumamos un número)

debugger; sirve para ver por consola uno a uno los pasos por los que pasa la interacción. en el apartado de sources del F12. es como verlo uno a uno manualmente.

quedaría así:
var numero = 100

for(var i = 0; i < numero; i++){
    console.log(i)
debugger;
}

bucle while se utiliza para hacer bucles de maneras infinitas. Aun así, contamos con un contador para ponerle fin en algún momento.
var year = 2018

while(year != 1991){
    console.log(year)
    year--;
}

bucle do while primero se ejecuta y despues comprueba la condicion.
var years = 40

do{
alert("tu tienes 40 años")
years++
}while(years < 50)
